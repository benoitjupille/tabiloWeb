TabiloWeb est un séquenceur utilisant Web Audio APi, CSS et jQuery.
J'ai aussi utilisé le plugin jQuery.knob.

Comment ça marche ?
. Le rond central est le "Master". Les autres ronds représentent des "Emetteurs" avec leur propre timing et paramètres. . Déplacez les "Emetteurs" vers le "Master" pour augmenter leur volume.
. Pour faire apparaître les paramètres des émetteurs, cliquez dessus.
. Jouez avec les paramètres des "Emetteurs" pour vous rendre compte de leurs effets.

Compatibilité : Chrome 30+, Safari 6+, Opera 15+
